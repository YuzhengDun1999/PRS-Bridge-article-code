
# PRS-Bridge-article-code

  

This is the viable code for the PRS-Bridge paper. 

## Folder description:

 - evaluation: contains code producing evaluation of each methods in real data analysis including getting $R^2$, transformed AUC, corresponding standard error, and relative efficiency.
 - plot-result: contains code producing figures in real data analysis.
 - run-methods: contains code running Lassosum, LDpred2, PRS-CS, and PRS-Bridge in real data analysis.
 - run-simulation: Contains code for running, evaluating, and visualizing the performance of Lassosum, LDpred2, PRS-CS, and PRS-Bridge in simulation studies.
